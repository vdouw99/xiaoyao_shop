安装向导
①环境要求
逍遥商城系统对服务器环境没有特殊要求，但必须满足以下基础安装条件
检查./data及其子目录777权限：{$mod_data}
检查./install目录777权限：{$mod_install}
检查./config.php文件777权限：{$mod_config}
②数据库配置
数据库服务器 端口
数据库用户名 密码
数据库名称
数据表前缀 默认是 "xy_"
③管理员帐号
管理帐号 网站管理员登录帐号
管理密码 网站管理员登录密码
恭喜！逍遥商城系统安装成功。

为保证网站数据安全，请立即删除./install目录
